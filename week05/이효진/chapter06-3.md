# 06-3 | 주성분 분석

### 차원과 차원 축소

**차원** : 데이터가 가진 속성 (특성)

**차원 축소** : 데이터를 가장 잘 나타내는 일부 특성을 선택해 데이터 크기를 줄이고 지도학습 모델의 성능을 향상시키는 방법

대표적인 차원 축소 알고리즘으로 주성분 분석( PCA )이 있음.

### 주성분 분석 (PCA)

: 데이터에 있는 분산(널리 퍼져있는 정도)이 큰 방향을 찾는 방법

**주성분** : 분산이 가장 큰 방향 벡터

**설명된 분산** : 주성분이 원본 데이터의 분산을 얼마나 잘 나타내는지 기록한 값

---

p.337 - 주성분 벡터의 화살표 위치는 의미가 없고(오른쪽 위, 왼쪽아래 상관없음) 분산이 큰 방향을 찾는 것이 중요함.

Q. 무슨 차이인지 모르겠음.

A.  **방향**이란 게 화살표가 가리키는 쪽(부호)이 아니라 **직선(line) 전체**를 뜻하는 것임.

(벡터의 부호 ≠ 방향)

↗ 방향  =  w
↙ 방향  =  -w

⇒상관없음.

공분산 행렬 CCC에 대해

Cw=λw

이면,

C(−w)=−Cw=λ(−w)

즉, 부호를 바꿔도 여전히 같은 고유값(=분산)을 가진다.

분산(variance)은 항상 0 이상인, 절댓값 개념의 크기
