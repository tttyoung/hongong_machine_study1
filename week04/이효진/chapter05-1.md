# 05-1 | 결정트리

**결정 트리** : 데이터를 여러 기준(질문)으로 나누면서 예측(분류)해가는 것. 리프 노드에서 가장 많은 클래스가 예측 클래스가 됨. **부모와 자식 노드 사이의 불순도 차이가 최대가 되도록 노드를 분할하고 마지막에 도달한 노드의 클래스 비율을 보고 예측을 만듦.**

### 불순도

**하나의 노드에 한 종류의 클래스만 들어있을수록 ‘순수’**  

**여러 클래스가 섞여 있을 경우 ‘불순’** 

1. 지니 불순도 = 1 - (음성 클래스 비율**2+양성 클래스 비율**2)

지니 불순도는  (순수) 0 ←——→ 0,5 (불순)

criterion 매개변수는 노드에서 데이터를 분할할 기준을 정함. 

결정 트리 모델은 부모 노드와 자식 노드의 불순도 차이가 가능한 크도록 트리를 성장시킴.

정보 이득 : 부모와 자식 노드 사이의 불순도 차이. 결정 트리 모델은 정보이득이 최대가 되도록 데이터를 나눔.

1. 엔트로피 불순도 (밑이 2인 로그를 사용해 계산)
