# 05-3 | 트리의 앙상블

### 정형 데이터와 비정형 데이터

정형 데이터 : 어떤 구조로 되어 있는 데이터.

앙상블 학습 : 정형 데이터에서 가장 뛰어난 성과를 내는 알고리즘

비정형 데이터 : 구조가 없는 데이터

### 앙상블

1. 랜덤 포레스트 :  결정 트리를 랜덤하게 만들어 결정 트리의 숲을 만들고 각 결정 트리의 예측으로 최종 예측을 만듦.

부트스트랩 샘플 : 중복을 허용하여 데이터를 샘플링하는 방식. 

1. 엑스트라 트리 : 랜덤 포레스트와 거의 동일하지만 부트스트랩 샘플을 사용하지 않고, 노드 분할을 최선이 아닌 랜덤으로 함.

랜덤 포레스트보다 속도는 빠르지만 많은 트리를 필요로 함.

많은 트리를 앙상블하므로 무작위로 분할해도 과대적합을 막고 검증세트의 점수를 높일 수 있음

1. 그레디언트 부스팅 : 깊이가 얕은 결정 트리를 사용하여 이전 트리의 오차를 보완하는 방식으로 앙상블하는 방법. 

성능은 뛰어나지만 앞선 방법들보다 속도가 느림.

깊이가 얕으면 과대적합에 강하고 일반화 성능이 높음.

1. 히스토그램 기반 그레이디언트 부스팅 : 입력 특성을 256개의 구간으로 나누어 사용해 노드 분할 속도가 매우 빠름.
