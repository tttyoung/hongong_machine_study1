# 7주차 과제

|  | DNN | CNN |
| --- | --- | --- |
| 입력 처리 방식 | 각 픽셀을 독립적으로 학습 → 공간적 정보 손실 | 필터를 통해 공간적 정보도 학습 |
| 훈련 속도 | 상대적으로 빠름 (연산 단순) | 상대적으로 느림 (합성곱 연산 다수 수행) |
| 성능 (정확도) | 이미지 패턴 인식에 약함 (일반적으로 정확도 낮음) | 이미지 특징 자동 추출 (정확도 높음) |

→ DNN이 CNN보다 훈련 속도 빠름

1. 구조적 특성
    
    CNN : 공간적 패턴 학습, DNN : 모든 입력 독립적 → 이미지 구조 정보 손실
    
2. 파라미터 효율성
    
    CNN은 필터를 재사용하기 때문에 같은 입력 크기에서도 파라미터 수가 훨씬 적음
    
3. 연산 방식 차이
    
    CNN 합성곱 연산은 연산량이 많아 속도 느리지만 병렬화 덕분에 효율적이며 결과적으로 정확도는 훨씬 우수
    
4. 데이터 표현 방식
    
    DNN : 1D 형태로 공간 정보가 손실됨, CNN : 2D 유지
